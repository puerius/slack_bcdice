# Slack BCDice

## WTF is this?
Dicebot for Slack which uses [BCDice-API](https://github.com/ysakasin/bcdice-api)

## Usage
1. Clone this repository.
1. Edit settings.rb
1. bundle install
1. bundle exec ruby ./bot.rb
